# SharwAPI Contracts Layer

[简体中文](/README_CN.md) | [English](/README.md)

This repository is the plugin interface layer (contracts layer) for [SharwAPI (also known as Sharw's API)](https://github.com/sharwapi/sharwapi.core).

To develop plugins, you need to reference this library. Please refer to the **[Online Documentation](https://sharwapi.hope-now.top)** for more details.
